Como Funciona

Veja como nossa API de recomendação pode transformar sua plataforma imobiliária.

1. Justificativa

Explica o raciocínio por trás da recomendação, considerando a localização do trabalho, escola e necessidades da família. 

"justificativa": "Considerando sua rotina, onde você trabalha no Bairro Barroca e seus filhos estudam no Bairro Betânia, o ideal é buscar um imóvel localizado em uma área intermediária, como o Barreiro ou Estoril, que facilite os deslocamentos diários. 

A sugestão é por uma casa espaçosa que ofereça privacidade e conforto para sua família, com um amplo quintal que permita a instalação de uma piscina ou que já venha equipada com uma."

2. Imóvel

Fornece detalhes específicos sobre o imóvel recomendado, incluindo tipo, localização e características. 

"imovel": {
"tipo": "Casa",
"localizacao": ["Barreiro", "Estoril"],
"cidade": "Belo Horizonte",
"estado": "MG",
"tamanho_m2": 250,
"quartos": 4,
"suites": 1,
"banheiros": 3,
"vagas_garagem": 3,
"quintal": {
  "espaco_para_piscina": true,
  "piscina_instalada": false
}
}

3. Diferenciais

Destaca as características únicas e vantagens do imóvel recomendado. 

"diferenciais": [
"Ambiente espaçoso para",
"privacidade e conforto",
"Localização estratégica",
"Área externa com lazer"
]

4. Preço Estimado

Fornece uma faixa de preço estimada para o imóvel recomendado. 

"preco_estimado": {
"min": 900000,
"max": 1500000
}

5. Proximidades

Lista os serviços e comodidades próximos ao imóvel recomendado. 

"proximidades": [
"Escolas",
"Comércios",
"Transporte público"
]
